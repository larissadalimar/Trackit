# TrackIt

Gerenciador de hábitos, onde se pode cadastrar e acompanhar hábitos.

## Para instalar as dependências

1. `npm install`

## Para rodar o projeto

1. `npm start`
2. Abrir a url [http://localhost:3000](http://localhost:3000).
